## Inputs

### Player 1

| Key           | Description    |
| ------------- | -------------- |
| `arrow up`    | move forward   |
| `arrow down`  | move backwards |
| `arrow left`  | rotate left    |
| `arrow right` | rotate right   |
| `space`       | shoot bullet   |

### Player 2

| Key | Description    |
| --- | -------------- |
| `w` | move forward   |
| `s` | move backwards |
| `a` | rotate left    |
| `d` | rotate right   |
| `q` | shoot bullet   |
